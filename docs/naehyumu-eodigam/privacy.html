시행일: 2026년 6월 10일

1. 처리하는 정보

내휴무어디감은 사용자가 선택한 근무표 사진에서 근무 일정을 읽어 앱 안의 캘린더와 위젯에 표시하는 앱입니다.

앱은 다음 정보를 기기 안에서 처리할 수 있습니다.

  • 사용자가 선택한 근무표 사진의 OCR 분석 결과
  • 근무일, 근무 유형, 출근 시간, 퇴근 시간, 업무 메모
  • 앱 내부 알림 설정

2. 사진 처리 방식

  • 근무표 사진은 OCR 분석에만 사용합니다.
  • 사진 원본은 앱에 저장하지 않습니다.
  • 사진 원본을 외부 서버로 전송하지 않습니다.
  • 사용자가 사진을 선택한 경우에만 해당 사진에 접근합니다.

3. 일정 데이터 저장 방식

  • 근무 일정은 사용자의 기기 내부 SwiftData 저장소에 저장됩니다.
  • iPhone 기본 캘린더에 저장하지 않습니다.
  • EventKit을 사용하지 않습니다.
  • 계정 생성, 로그인, 서버 동기화를 제공하지 않습니다.

4. 알림 처리 방식

앱은 사용자가 허용한 경우 다음 목적의 로컬 알림을 예약할 수 있습니다.

  • 전날 밤 근무 안내
  • 출근 전 근무 안내
  • 사용자가 직접 누른 테스트 알림

알림은 기기 안에서 예약되는 로컬 알림이며, 원격 푸시 서버를 사용하지 않습니다.

5. 외부 전송 및 제3자 제공

현재 버전의 앱은 근무표 사진, OCR 결과, 근무 일정, 알림 설정을 외부 서버로 전송하지 않습니다. 또한 광고, 트래킹 SDK, 제3자 분석 SDK를 사용하지 않습니다.

6. 사용자의 선택권

사용자는 iOS 설정에서 언제든지 사진 접근 권한과 알림 권한을 변경할 수 있습니다. 앱 삭제 시 기기 내 앱 데이터도 iOS 정책에 따라 삭제됩니다.

7. 아동 개인정보

내휴무어디감은 만 13세 미만 아동을 대상으로 하지 않습니다.

8. 변경 사항

개인정보 처리방침이 변경되는 경우 앱 또는 배포 페이지를 통해 안내합니다.

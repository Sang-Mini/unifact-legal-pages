# Unifact Legal Pages

App Store Connect에 입력할 앱별 법적 문서 정적 사이트입니다.

## 현재 앱

### 내휴무어디감

- 개인정보처리방침: `docs/naehyumu-eodigam/privacy.html`
- 고객지원: `docs/naehyumu-eodigam/support.html`
- 이용약관: `docs/naehyumu-eodigam/terms.html`
- 법적 고지: `docs/naehyumu-eodigam/legal.html`

## GitHub Pages 설정

1. GitHub에서 `unifact-legal-pages` 저장소를 만듭니다.
2. 이 폴더를 해당 저장소에 push합니다.
3. Repository Settings > Pages로 이동합니다.
4. Source: `Deploy from a branch`
5. Branch: `main`, Folder: `/docs`
6. 저장 후 배포 URL이 열리는지 확인합니다.

## App Store Connect 입력 URL 예시

현재 저장소 기준 App Store Connect 입력 URL:

- 개인정보처리방침: `https://sang-mini.github.io/unifact-legal-pages/naehyumu-eodigam/privacy.html`
- 고객지원: `https://sang-mini.github.io/unifact-legal-pages/naehyumu-eodigam/support.html`

실제 URL은 GitHub Pages가 표시하는 주소 기준으로 입력하세요.
